GUESSING THE NUMBER

    INSTRUCTIONS

  • The user will enter a number between 1 and 100. If user enters a invalid number, show a error.
  • The computer will have 5 chances to guess the number.
  • If the computer correctly guesses the number within 5 attempts, it wins
  • If the computer is unable to guess the number, the user wins.
